## 1순위 상세 — 어트랙트 라이브 미러 설계 초안

```
유휴 45초 → AttractOverlay:
  카메라 프레임 + 지나가는 모든 사람 스켈레톤(num_poses=2 warm 모델 재사용)
  + 큰 문구 "지나가다 멈추면 게임 시작!" / 랜덤 간격 플러시(링 확산 + 효과음)
  + 사람이 3초 이상 화면 중앙에 머물면 → "손을 들면 시작!" → 반응속도 게임 자동 진입
터치/키 입력 시 기존처럼 홈으로
```
- 성능: warm estimator 공유(추가 모델 로드 없음), 어트랙트 중에만 워커 가동
- 프라이버시: 영상 저장 없음(현 구조 그대로), 안내 문구 표시 권장
- 검증: `verify_games.py` 패턴으로 오프스크린 스크린샷 + "사람 감지→유도 문구 전환" 상태 어서션

## 가설 단계 (근거 미확보 — 소규모로 실험 후 판단)

리서치에서 리더보드/경쟁 심리 주장들은 검증 임계를 못 넘었다. 아래는 통념 기반 가설로,
행사에서 관찰(재도전율, 체류)로 자체 검증할 것:
- 일간/행사 단위 리더보드 리셋 (1위가 고착되면 신규 도전 의욕 하락 가설)
- "지금 1위와 N점 차" 근접 목표 표시
- 점수대 칭호("번개 반사신경", "인간 용수철") — 사진 카드와 결합 시 공유 가치
- 즉석 토너먼트(연속 도전 즉석 순위) — versus 인프라 재사용

## 추천 로드맵

1. **행사 전 필수(1주)**: ①어트랙트 라이브 미러 + 랜덤 플러시 ②운영 체크리스트 문서화 ③홈 카드 순서 조정
2. **여유 시(1주)**: ④원거리 가독성 패스 ⑤호기심 장치(근접 목표 + 보너스 라운드)
3. **행사 후 데이터로**: ⑥사진 카드+QR ⑦리더보드 가설 실험
