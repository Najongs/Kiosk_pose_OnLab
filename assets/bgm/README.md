# 배경음악 (BGM)

이 폴더에 음악 파일(`.mp3` `.wav` `.ogg` `.m4a` `.flac`)을 넣으면
앱 시작 시 자동으로 반복 재생됩니다. 여러 개면 순서대로 돌아갑니다.

- 켜기/끄기: 관리자(⚙) → **배경음악** 체크
- 볼륨은 낮게(25%) 재생되어 효과음을 가리지 않습니다.

## 저작권 무료 음원 구하기
- https://pixabay.com/music/  (상업적 사용 가능, 출처표기 불필요)
- https://incompetech.com/music/  (CC-BY, 출처표기 필요)
- 유튜브 오디오 라이브러리 (스튜디오 → 오디오 보관함)

키오스크 환경이므로 잔잔한 루프형 트랙(lo-fi, ambient)을 권장합니다.
