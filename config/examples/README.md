# 동작 예시 이미지

세션 중 화면 **왼쪽**에 "따라해 보세요" 목표 자세 예시로 표시됩니다.

- 파일명 = 자세 이름 + 확장자: `<pose>.png|jpg|jpeg|webp`
  (자세 이름은 `config/poses/*.json` 의 파일명과 동일: `forward_bend`, `side_bend`,
   `overhead_reach`, `tpose`, `one_leg_balance`)
- 세로형(사람 전신) 이미지가 박스에 잘 맞습니다. 배경이 단순할수록 좋습니다.
- 없으면: 관리자에서 캡처한 참조 스켈레톤 → 그것도 없으면 "예시 준비 중" 표시.

> 현재 들어있는 이미지는 임시 예시(요가 일러스트)입니다. 실제 동작에 맞는
> 사진/GIF 첫 프레임으로 교체하세요. `tpose` 예시는 아직 없습니다(추가 권장).
